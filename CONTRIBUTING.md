# 🤝 Contribution Guidelines

Όλες οι συνεισφορές, αναφορές σφαλμάτων (bug reports), διορθώσεις σφαλμάτων (bug fixes), βελτιώσεις τεκμηρίωσης (documentation improvements), επεκτάσεις (enhancements) και ιδέες είναι ευπρόσδεκτες.

---

## 🚀 Πώς να Συνεισφέρετε

*(Προαιρετικά: Μπορείς να προσθέσεις τα βασικά βήματα για συνεισφορά, χρησιμοποιώντας όσα μάθαμε προηγουμένως, π.χ.:)*

1.  **Fork** το αποθετήριο.
2.  Δημιουργήστε έναν κλάδο (branch) για το νέο σας χαρακτηριστικό: `git checkout -b feature/your-feature-name`.
3.  Κάντε τις αλλαγές σας και **Commit** με περιγραφικά μηνύματα.
4.  Κάντε **Push** τον κλάδο σας στο GitHub.
5.  Ανοίξτε ένα **Pull Request** στον `main` κλάδο.
