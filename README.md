# 💰 Simple Interest Calculator (Υπολογιστής Απλού Τόκου)

Ένα απλό εργαλείο για τον υπολογισμό του απλού τόκου (Simple Interest) που κερδίζεται ή πληρώνεται επί ενός κεφαλαίου για μια συγκεκριμένη χρονική περίοδο.

---

## 🎯 Σκοπός

Αυτός ο υπολογιστής επιτρέπει στους χρήστες να προσδιορίσουν γρήγορα το συνολικό ποσό τόκου που παράγεται από μια επένδυση ή ένα δάνειο, βάσει της βασικής φόρμουλας απλού τόκου.

## ⚙️ Φόρμουλα Υπολογισμού

Ο απλός τόκος (Simple Interest, $I$) υπολογίζεται με την ακόλουθη μαθηματική φόρμουλα:

$$I = P \cdot r \cdot t$$

Όπου:

* **$I$** (Interest): Το συνολικό ποσό τόκου που παράγεται.
* **$P$** (Principal Amount): Το αρχικό κεφάλαιο (το ποσό του δανείου ή της επένδυσης).
* **$r$** (Annual Interest Rate): Το ετήσιο επιτόκιο (ως δεκαδικός αριθμός, π.χ., το 5% γράφεται ως 0.05).
* **$t$** (Time): Ο χρόνος σε έτη (π.χ., 3 χρόνια, 6 μήνες γράφεται ως 0.5 έτος).

---

## 🛠️ Τρόπος Χρήσης

Για να χρησιμοποιήσετε τον υπολογιστή, πρέπει να εισάγετε τις ακόλουθες τρεις τιμές:

1.  **Κεφάλαιο ($P$):** (π.χ., 1000)
2.  **Ετήσιο Επιτόκιο ($r$):** (π.χ., 0.05 για 5%)
3.  **Χρόνος σε Έτη ($t$):** (π.χ., 2)

### Παράδειγμα:

Αν εισάγετε:
* $P = 1000$
* $r = 0.05$
* $t = 2$

Ο υπολογιστής θα επιστρέψει το αποτέλεσμα:

$$I = 1000 \cdot 0.05 \cdot 2 = 100$$

### Συνολικό Ποσό ($A$):

Το συνολικό ποσό που θα επιστραφεί ή θα λάβετε στο τέλος ($A$) υπολογίζεται ως:

$$A = P + I$$

Στο παράδειγμα: $A = 1000 + 100 = 1100$
